Running the program:
Program may be opened using F5 or F6 buttons within Atom.
Please ensure the button used launches Python 3.

#####ABOUT THE PROGRAM######

Program is an attempt at a basic drawing program using Turtle.
User can click and drag turtle to make a line/shape, or click anywhere
on the screen to select a different starting point.
User can use right click to clear contents of the screen.
User can select colors by using buttons: 
{('1', "Black"), ('2', "Red"), ('3', "Orange"), ('4', "Green")}

Instructions are printed upon initialisation of screen, and should be
cleared before drawing.
